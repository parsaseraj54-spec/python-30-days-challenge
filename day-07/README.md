# Python Mini Functions Project 🐍

A simple Python project using functions.

This program includes:

- ➕ Add two numbers
- 🔢 Check if a number is even or odd
- 🧠 Practice creating and using functions in Python

---

## Features

### 1. Add Numbers
The user enters two numbers and the program returns their sum.

Example:

Input:
Enter First Number: 10
Enter Second Number: 20

Output:
30


### 2. Check Even Number
The user enters a number and the program checks whether it is even.

Example:

Input:
Enter a Number: 8

Output:
True

---

## Code Structure

```python
def add(a, b):
    return a + b

def is_even(a):
    return a % 2 == 0
```
