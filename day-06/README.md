# 🔢 Find Maximum Number in a List

A simple Python program that finds the largest number in a list using a loop and conditional statements.

## 📌 Features

- Stores numbers inside a list
- Loops through all values
- Compares numbers one by one
- Finds and prints the maximum value

## 💻 Code

```python
numbers = [7, 89, 34, 26]

max_numbers = numbers[0]

for num in numbers:
    if num > max_numbers:
        max_numbers = num

print(max_numbers)
```

## ▶️ Output

```bash
89
```

## 🧠 Concepts Used

- Lists
- Variables
- For Loop
- If Statement
- Comparison Operators

## 🎯 Purpose

This project was built for practicing Python fundamentals and understanding how maximum value searching works without using built-in functions like `max()`.
