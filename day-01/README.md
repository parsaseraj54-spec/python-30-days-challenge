# Name Age Calculator

A simple Python program that calculates how many years are left until the user turns 30, or how many years ago they turned 30.

## How it works:
The program prompts the user for their name and age. It then uses conditional logic (if/elif/else) to determine the appropriate message to display.

## Concepts Used:
*   **Variables**: Storing user input (name, age) and calculation results.
*   **Input Function**: Getting data from the user.
*   **Type Casting**: Converting the age input from a string to an integer using `int()`.
*   **Conditional Statements**: Using `if`, `elif`, and `else` to handle different age scenarios.
*   **f-strings**: For clear and formatted output, combining text and variables.
*   **Basic Arithmetic Operators**: For calculating years left or years passed.

## How to Run:
1.  Save the code as a Python file (e.g., `calculator.ipynb`).
2.  Open your terminal or command prompt.
3.  Navigate to the directory where you saved the file.
4.  Run the script using: `python calculator.py` (or `python3 calculator.py`)
5.  Follow the prompts to enter your name and age.
