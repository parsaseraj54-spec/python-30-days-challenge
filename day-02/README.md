#🧮 Even or Odd Number Checker
A simple Python program that checks if a number entered by the user is Even or Odd.

##🚀 How it works:
The program asks the user to enter a number. It then checks the remainder when divided by 2:

If the remainder is 0, the number is Even.
Otherwise, the number is Odd.
##🧠 Concepts Used:
Variables: Store the user’s input number.
Input Function: input() to collect data from the user.
Type Casting: Convert input from string to integer using int().
Conditional Statements: Use if and else to determine even or odd.
Modulo Operator (%): To find the remainder after division.
##💻 Code Example:
python
number = int(input('Enter Your Number: '))

if number % 2 == 0:
    print('Even')
else:
    print('Odd')
##⚙️ How to Run:
Save the code in a file, for example even_odd.py.
Open your terminal or command prompt.
Navigate to the folder where the file is saved.
Run the program:
text
   python even_odd.py
Enter a number when prompted and the program will tell you if it’s even or odd.
🎯 Why is this useful?
Checking if a number is even or odd is a classic beginner step in learning programming logic and conditionals.
